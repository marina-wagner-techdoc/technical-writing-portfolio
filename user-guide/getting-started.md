# Getting Started – fitElle App

## Overview

Fitelle is a fitness and wellness app designed to help users track workouts, nutrition, and progress.

## Getting Started

1. Download the fitElle app
2. Create an account
3. Set your personal goals
4. Start tracking your activities

## Features

- Workout tracking
- Progress monitoring
- Personalized goals

## Cycle Tracking

Track your cycle by entering daily data in the app.

../assets/cycle.png

The cycle tracking screen allows users to log symptoms, track phases, and monitor patterns over time.

## Tips

- Update your data regularly
- Use consistent inputs for better tracking
